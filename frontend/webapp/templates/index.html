About MLOps Assignment 2

Automated Pipeline

Features a complete CI/CD/CT pipeline using GitHub Actions. The model is automatically retrained on new code pushes using the Kaggle Cats vs Dogs dataset.

Transfer Learning

Utilizes MobileNetV2, a state-of-the-art lightweight neural network, fine-tuned for high accuracy and fast inference on edge/cloud devices.

Secure & Scalable

Deployed on Azure Kubernetes Service (AKS) with production-grade Ingress and HTTPS (Let's Encrypt) security.

Full Observability

Integrated with Prometheus & Grafana for real-time monitoring of inference latency, request counts, and system health.
